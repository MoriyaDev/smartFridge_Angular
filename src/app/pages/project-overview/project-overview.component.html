🚀 יצירת הפרויקט

יצרתי קומפוננטות עיקריות כמו Navbar, HomePage, Footer, וחילקתי למודולים נפרדים.

🗺️ ניתובים (Routing)

הגדרתי נתיבים כמו עמוד בית, רשימת מוצרים, ודף פרטי מוצר + טעינה עצלה לשיפור ביצועים.

📊 עבודה עם נתונים

השתמשתי ב-Services לניהול מוצרים, מקרר ופתקים, עם קריאות GET/POST לשרת.

📝 טפסים ואינטראקציה

בניית טפסים עם Reactive Forms, ולידציות, ושמירה על עיצוב מינימליסטי ונקי.

🛡️ אבטחה ושיפורים
